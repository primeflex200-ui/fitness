## Animation Sequence

### Adding Food (1 second)
```
0ms:   User types food name
100ms: User clicks Add
200ms: Food appears in list
300ms: Toast notification shows
1000ms: Toast fades out
```

### Generating Plan (5-15 seconds)
```
0ms:   User clicks Generate
100ms: Loading spinner starts
500ms: API call sent
5000ms: Response received
5100ms: Plan displayed
5200ms: Day selector shown
```

### Switching Days (200ms)
```
0ms:   User clicks day button
50ms:  Button highlights
100ms: Meals fade out
150ms: New meals fade in
200ms: Animation complete
```
